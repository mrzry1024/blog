    starctf2022/crypto-ezRSA at main · sixstars/starctf2022 (github.com) (opens new window)

    sqrt(n) for high bits. Use xor relationship to get middle bits. Partial p to get the rest.

    # 解题

    在题目中， q=next_prime(p^((1<<900)-1)^getrandbits(300)) ，由于 和 相近，对 开方取高位即可得到 的高位，而中间的899位则是与1异或，如果求出中间部分，那么便可以进行已知p高位攻击求出 了

    对于中间部分，先看看下面这个小例子

    p = '101011'
q = '010100'
print(int(p, 2)*int(q, 2))  # 860

# p和q赋值如下
p = '111111'
q = '000001'

# 第一位异或后
p = '011111'
q = '100001'
print(int(p, 2)*int(q, 2))  # 1023
'''
大于原先值，那么该位不用改，即
p = '111111'
q = '000001'
'''

# 第二位异或后
p = '101111'
q = '010001'
print(int(p, 2)*int(q, 2))  # 799
'''
小于原先值，那么该位要改，进行异或，即
p = '101111'
q = '010001'
'''

# 对n开方得到p的高位
pm = int(gmpy2.iroot(n, 2)[0]) >> 900 << 900

# 异或关系得到p中间几位
p = pm + (1<<900)-1
q = pm + (1<<(300-1))

for i in range(899, 300, -1):
    cur = 1<<i
    if (p^cur) * (q^cur) < n:
        p ^= cur
        q ^= cur
print(p)
